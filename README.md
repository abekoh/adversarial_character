# adversarial_character
## Requirement
- Python 3.5
- deap 1.0.2
- Keras 1.2.2
- Tensorflow 0.12.1
- Numpy
- Scipy
- Pillow
- h5py
- imageio

## Setup
```
pip install deap
pip install keras==1.2.2
vim ~/.keras/keras.json # change "theano" to "tensorflow"
pip install tensorflow==0.12.1 # or tensorflow-gpu==0.12.1
pip install pillow
pip install h5py
pip install imageio
```
