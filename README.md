# adversarial_character
## Requirement
- Python 3.6.1
- deap 1.1.0
- Keras
- Tensorflow
- Numpy
- Pillow
- h5py

## Setup
```
chmod +x install_deap.sh
sh ./install_deap.sh
pip install keras==1.2.2
vim ~/.keras/keras.json # change "theano" to "tensorflow"
pip install -U tensorflow # or tensorflow-gpu
pip install Pillow
pip install h5py
```
