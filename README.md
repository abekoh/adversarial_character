# adversarial_character
## Requirement
- Python 3.5
- deap 1.0.2
- Keras 1.2.2
- Tensorflow 1.0.0
- Numpy
- Pillow
- h5py
- imageio

## Setup
```
pip install deap==1.0.2
pip install keras==1.2.2
vim ~/.keras/keras.json # change "theano" to "tensorflow"
pip install tensorflow==1.0.0
pip install pillow
pip install h5py
pip install imageio
```
